Do que se trata?


Este projeto visa utilziar a API (Application Programming Interface (Interface de Programação de Aplicação)), ou seja, um recurso que através de uma URL, "capta" informações que foram salvas em algum banco de dados.

Sendo assim, somos capazes de receber as informações da plataforma do GitHub e implementa-las em nosso projeto.
